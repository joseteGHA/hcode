Que es BootStrap

Es un framework web de código abierto para el diseño sitios y aplicaciones web

Es un framework creado Mark Otto y Jacob Thornot de Twitter

Se liberó en el agosto 2011

Y en 2012 fue el proyecto más popular de Github

Su versión actual es la 4.

Como máquetamos nuestra web con BootStrap

Grid System BootStrap

Container

Es el contendor donde introduciremos todos nuestros elementos de BootStrap


[div class="container"]
[div class="container-fluid"]

Row

Corresponde a una fila donde podemos introducir las columnas.


[div class="container"]
    [div class="row"]
                            ....

Col

Sistema de columnas BootStrap divide las pantallas en 12 columnas


[div class="container"]
  [div class="row"]
    [div class="col"]
      [p]Columna 1[/p]
    [/div]
    [div class="col"]
      [p]Columna 2[/p]
    [/div]

Podemos establecer y maquetar los límites de las diferentes secciones 


[div class="container"]
    [div class="row"]
        [div class="col-6"]
            [p]Columna 1[/p]
        [/div]
        [div class="col-6"]
            [p]Columna 2[/p]
        [/div]

Inherentemente nos permite crear web responsive

Dispone de puntos de rotura

Anchura de columna en función del contenido.

col-md-auto

Sistemas de alineación del contenido


[div class="container"]
    [div class="row align-items-start"]
        [div class="col"]
    ...
    [div class="row align-items-center"]
    ...
    [div class="row align-items-end"]

Offseting columns

Para mover columna derechas.


[div class="row"]
    [div class="col-md-4"].col-md-4[/div]
    [div class="col-md-4 offset-md-4"].col-md-4 .offset-md-4[/div]
[/div]
