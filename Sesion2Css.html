Historia del CSS

¿Que es CSS?

CSS es la abreviatura empleado para referirse a Cascade Style Sheets(Hojas de Estilo en Cascada)

Con el uso de CSS se le índica al documento el estilo de cada elemento en la página a traves de un archivo externo o directamente en el código HTML

Historia del CSS

El CSS ha evolucionado al igual que el HTML

W3C Mantiene el estandar.

En el año 1995 se crea la primera versión.

En el año 1998 se publica la segunda especificación

En el año 2009 se publica la especificación 2.1

En la actualidad estamos en la especificación 3.0

Otras tecnologías que se han desarrollado a partir de CSS

  • Sass
  • Lesscss
  • PostCSS

Que podemos hacer con CSS3

  • Estructura y maquetar el diseño completo del sitio Web.
  • Generar una estructura modular para los estilos
  • Crear transacciones interacciones simples
  • Crear objetos gráficamente sin incluir imágenes que ensucien el HTML
  • Generar entornos aptos para móviles.

Qué evitar.

  • Limitarse a formatear textos u objetos
  • Utilizar siempre la etiqueta sytle o el atribo del mismo nombre en el HTML
  • Repetir definiciones de caractersíticas generales.

Lo que hay que tener en cuenta del CSS

  • Todos los objetos son cajas.
  • Funciona a traves de selectores.
  • Los objetos son modificados con propiedades.
  • Hay propieades/selectores incompatibles con algunas navegadores.
  • Las propieades se heredan de padres e hijos.

DOM(DOCUMENT OBJECT MODEL)

Cuando una página Web se carga en el navegador Web se crea lo que se conoce como DOM, y se construye como un árbol de objetos.

Es a través de este árbol que podemos aplicar estilos a cada uno de los elementos de nuestra página HTML y a través de CSS aplicar un diseño específico a cada elemento. Esto lo haremos a traés de selectores

SELECTORES EN CSS

Selectores de Etiqueta

Selectores ID:

  • Estos selectores se utilizar para especificar el estidlo de un elemento único en la página Web actual
  • Para definir un ID de CSS se utiliza el operador #
  • SE debe aplicar el atributo id en los elementos

Selectores Class:

  • Estos selectores se utilizan para especificar el estilo de un grupo de elementos
  • Para definir una clase CSS se utilizar el operador punto (.)
  • Se debe aplicar el atributo class en los elementos HTML

Formato de Texto

Como dar formato al texto

Propiedad color

Se puede establecer en formato hexadecimal o rgba

color:#ff5bc3;

Propiedad font-family

Defin la familia tipográfica

font-family:"Avant Garde CE",Helveltica, sans-serif;

Propiedad font-size

Define el tamaño del texto (px, em)

font-size: 50px;

Propiedad font-weight

Establece el peso/grosor del texto

Puede ser bold, normal o el valor en puntos de 100 a 900)

font-weight:bold;

Propiedad font-sytle

Cambia el estilo de texto

Puede ser: normal, italic, oblique

font-style:italic;

Propiedad text-transform

Fuerza la utilización de un estilo de escritura

Valores posibles:uppercase, lowercase, capitalize

text-transform:uppercase;

Propiedad text-decoration

Define la decoración del texto

Valores posibles: none, underline, overline y line-through

text-decoration:underline;

Propiedad text-indent

Genera un sangrado para el texto

Sus valores pueden ser positivos o negativos

text-indent:20px;

Propiedad text-align

Alinea el exto de la caja

Sus valores posibles: left, right, center, justify

text-align:right;

Propiedad letter-spacing

Genera una seraparación entre carácteres.

Pueden ser también positivos o negativos

letter-spacing:3px;

Propiedad line-height

Aumenta o reduce el interlineado de un texto

line-height:30px;

Trabajando con los fondos de nuestros elementos.

Propiedad background-color

Define el color de fondo para el objeto

Puede ser hexadecimal o rgba

background-color: #3f9ff5

Propiedad background-image

Establece una imagen de fondo para el objeto

background-image:url(../images/fondo.jpg);

Propiedad background-repeat

Define la forma de repetición del fondo

Sus valores pueden ser: repeat, repeat-y, repeat-x y no repeat

background-repeat: repeat-y;

Propiedad background-position

Índica la posición del fondo

Sus valores son: center, top, left, right, bottom y coordenadas x,y

background-position: top center;

Propiedad background-size

Define el tamaño del fondo del objeto

background-size:cover;
